## ヤマト

### ステータスの種類

- picked: 荷物受付, 発送, 荷受け
- delivering: 作業店通過, 配達店通過, 依頼受付（～～）, 投函予定, 保管中, 配達中, 配達予定
- absence: 持戻（～～）
- completed: 配達完了, 投函完了
- issue: 調査中, 返品

### 有効な追跡番号

- 11-12 桁の数字のみ、最後の一桁はチェックディジット(10-11 桁 % 7)

## 日本郵便

### ステータスの種類

- picked: 集荷, 引受
- delivering: 中継, 通過, 到着, 発送, 持ち出し中, 配達希望受付, ～～送付, ～～保管, 転送
- absence: ご不在のため持ち戻り
- completed: お届け先にお届け済み, 窓口でお渡し
- issue: ～～返送, 調査中

### 有効な追跡番号

- 11-12 桁の数字、最後の一桁はチェックディジット(10-11 桁 % 7)
